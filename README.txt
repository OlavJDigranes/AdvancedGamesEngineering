README

Application Usage:
    - Launch Application    
    - Input values
        - Absolute magnitue in range of -10 to 20
        - Spectral Type: O, B, A, F, G, K, M
        - Number of planets: A positive integer above 0
    - Click "Start"
    - In simulation scene:
        - Zoom in and out by scrolling
        - Move camera between celestial bodies using T and G keys on keyboard
        - Spawn asterid using R key on keyboard
        - Exit simulation and application using ESC
    - Re-Run by repeating above steps
